# Research

This folder stores short structured research notes.

## Purpose
- keep external references in one place
- capture inspiration without mixing it into live design truth
- store technical notes discovered outside the codebase

## Structure
- `research-template.md`: reusable note format
- research entries named by date and topic

## Naming
- `2026-04-20-phaser-timer-notes.md`
- `2026-04-20-platform-fighter-reference.md`

## Required Sections
- topic
- source or reference
- takeaway
- relevance to project

## Rules
- keep entries short
- separate source facts from project interpretation
- do not document an external idea as a live feature unless it exists in code
- if research leads to a concrete code or doc change, record that later in `docs/revisions/`

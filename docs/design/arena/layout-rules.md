# Arena Layout Rules

## Current Rules

- keep the stage readable on one screen
- preserve left/right fairness
- preserve ring-out threat
- support the current three fighter fantasies
- avoid clutter beyond the existing platform count

## Current Special Rule

The stage already uses movement and final-phase mutation.
Do not add more layout gimmicks unless older visual/dead code is removed first.

## Readability Rule

Platforms must remain easy to parse against the background.
The world fiction can deepen, but platform readability cannot be traded away.

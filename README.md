# Pixel Brawl

`Pixel Brawl` is a local `1v1` Phaser 3 platform fighter built for the Platanus arcade contest.
The project is constrained by:

- Phaser 3 only
- no external assets
- code-generated visuals and audio only
- no network calls
- under `50 KB` after minification

The current playable loop already includes:

- menu
- character select
- live match
- timer and countdown
- final phase
- percent and stamina systems
- pickups and buffs
- procedural music / SFX
- KO, respawn, and end screen
